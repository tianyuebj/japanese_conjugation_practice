# 日语单词变形练习 Anki 插件

在 Anki 复习日语单词时，针对当前单词进行变形选择题练习。
<img width="1920" height="1040" alt="image" src="https://github.com/user-attachments/assets/40253096-6783-4c7f-b80d-201d5ba24607" />


## 功能

- **一键启动**：复习界面底部显示"练习变形"按钮，名词卡片自动显示"无需变形"（禁用状态）
- **选择题练习**：一次练习一个单词的所有变形，4 选 1 卡片式按钮
- **即时反馈**：答对绿色高亮，答错红色标出，并显示变形规则说明
- **朗读功能**：选择选项后自动朗读该选项（需 Windows 日语语音包）
- **进度追踪**：顶部进度条 + 实时正确率
- **变形总结**：完成后展示所有变形形式及得分

### 支持的变形类型

| 词性 | 变形 |
|------|------|
| 动词（五段、一段、サ变、カ变）| ます形、て形、た形、ない形、可能形、受身形、使役形 |
| い形容词 | て形、过去形、否定形、副词形 |

## 安装

1. 下载 `japanese_conjugation_practice` 文件夹
2. 复制到 Anki 插件目录：
   - Windows：`%APPDATA%\Anki2\addons21\`
   - Mac：`~/Library/Application Support/Anki2/addons21/`
   - Linux：`~/.local/share/Anki2/addons21/`
3. 重启 Anki

### 朗读功能（可选）

需要 Windows 系统安装日语语音包：

`设置 → 时间和语言 → 语音 → 添加语音` → 搜索「日本語」安装（推荐 Microsoft Haruka）

## 使用方法

1. 在 Anki 中开始复习日语单词
2. 点击底部"练习变形"按钮，或按 `Ctrl+J`
3. 点击选项后会朗读该选项读音，再点"确认答案"
4. 完成所有题目后查看变形总结

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+J` | 启动练习（复习界面） |
| `Enter` | 确认答案 / 下一题 |
| `Esc` | 退出练习 |

## 系统要求

- Anki 2.1+
- Windows（朗读功能依赖 Windows SAPI，其他平台朗读功能不可用）
